
#  UniServe

UniServe is an AI-powered peer-to-peer campus delivery platform where students can get food and essentials delivered within hostels while other students earn through micro-gigs.

---

## ✨ Key Features

* 🔐 University email-based login
* 📦 Order from outlets, gates, or anywhere on campus
* 👥 Choose your delivery partner 
* 💬 Real-time chat with delivery partner
* ⭐ Ratings & reviews system
* 🧠 Smart matching based on rating, proximity & performance

---


## 🛠️ Tech Stack

* Frontend: React
* Backend: Node.js + Express
* Database: MongoDB
* Real-time: Socket.io
* Payment: Razorpay
